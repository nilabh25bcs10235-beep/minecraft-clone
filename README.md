# 🌲 Minecraft Clone - Browser Edition

A fun, playable browser-based Minecraft-inspired game built with pure HTML5 Canvas and JavaScript. No external dependencies — just open `index.html` and play!

## ✨ Features
- **2D Top-Down Voxel World**: Explore a grid of blocks (Grass, Dirt, Stone, Sand, Wood, Water)
- **Break & Place Blocks**: Left-click to mine, right-click to place
- **Player Movement**: Smooth WASD / Arrow key controls
- **Hotbar**: 6 block types — switch with number keys (1-6) or click
- **Creative Mode**: Unlimited blocks, focus on building and exploring
- **Nice UI**: Hotbar, instructions, coordinates, crosshair
- **Procedural-ish World**: Grassland with stone patches and simple trees

## 🎮 How to Play
1. Open `index.html` in Chrome, Firefox, Edge, or any modern browser.
2. Click on the canvas to start.
3. **Move**: `W A S D` or Arrow Keys
4. **Break block**: Left Mouse Button (must be adjacent or close)
5. **Place block**: Right Mouse Button
6. **Select block type**: Press `1` to `6` or click the hotbar icons
7. Build, mine, and have fun!
